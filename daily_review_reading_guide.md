# A 股主线研究日报阅读指南

本指南用于说明如何阅读《A 股主线研究日报 V0.3》，让它真正服务于你的原始目标：

> 经过一段时间观察和复盘，找到市场环境良好时，具有持续性的中高级别主线板块。

请先记住一句话：

**这份日报不是个股推荐器，不是择时器，也不是买卖信号。它是用来识别“市场是否适合研究主线、哪些主线正在形成、哪些主线正在退潮”的研究闭环工具。**

## 1. 每天先看哪三件事

打开日报后，不要先看个股，也不要先看涨幅最大的行业。

建议按这个顺序看：

1. **市场环境**
   先判断今天的大盘背景是支持主线扩散，还是只适合观察。

2. **主线变化复核**
   再看哪些行业是延续、晋级、降级、分歧、退潮或重新升温。

3. **退潮与风险**
   最后确认哪些方向需要降频观察，避免被单日反弹误导。

如果这三件事没有形成清晰结论，后面的载体池只能作为资料库，不要当成行动清单。

## 2. 今日结论卡片怎么读

“今日结论卡片”是日报的压缩摘要。

它主要回答三个问题：

- 市场现在偏进攻、偏观察，还是偏防守？
- 当前比较值得继续跟踪的主线大概是谁？
- 哪些方向已经出现退潮或风险复核信号？

这里的“B 级主线”“退潮警报”只是研究优先级，不代表可以直接操作。尤其当市场环境分不高时，即使有 B 级主线，也要理解为“值得继续观察其持续性”，而不是“马上行动”。

## 3. 市场环境怎么读

市场环境是整份日报的第一层约束。

它不决定“是否研究”，但决定“研究结论的力度”。

| 环境分 | 状态 | 你应该怎么理解 |
|---:|---|---|
| 0-29 | 防守/弱势 | 市场整体弱，主要看风险和退潮，不要急着找机会 |
| 30-44 | 弱观察 | 有局部修复，但整体还弱，重点看谁能撑住 |
| 45-54 | 偏观察 | 有结构性方向，但不足以说明行情全面扩散 |
| 55-69 | 中性偏强 | 主线可以认真跟踪，但仍要看延续和分歧 |
| ≥70 | 进攻 | 环境支持主线扩散，但仍需看行业生命周期 |

阅读重点不是分数本身，而是：

- 今天比昨天变强还是变弱？
- 上涨股票比例是否提升？
- 站上 MA20 的比例是否改善？
- 20 日收益为正的股票比例是否扩大？
- 成交额是否支持扩散？

一个很常见的误区是：看到当天上涨股票很多，就以为市场转强。日报会把“短线情绪修复”和“中期赚钱效应改善”分开看。前者可能只是一日反弹，后者才更接近主线行情的土壤。

## 4. 主线总览怎么读

主线总览是“当前市场结构地图”。

你不需要每天把所有字段都研究一遍，先看四列就够：

- **级别**
- **状态**
- **20 日 / 60 日收益**
- **宽度**

### 级别

| 级别 | 含义 |
|---|---|
| A | 高优先级主线，强度、宽度和生命周期更完整 |
| B | 可持续研究的主线，但可能仍有瑕疵 |
| C | 升温、修复或弱势反弹方向，需要继续验证 |
| 退潮 | 已经明显转弱，主要做风险复核 |
| 低频监控 | 暂时不是核心方向，只保留观察 |

不要机械理解为“A 一定好、C 一定差”。有些 C 级是新升温，后续可能晋级；有些 B 级可能是防御抱团，不一定是进攻主线。

### 状态

状态比级别更重要。

常见状态可以这样理解：

- **真延续**：不是单日脉冲，过去一段时间有持续性。
- **峰值回撤后修复**：此前强过，后来回撤，现在在修复，需要确认是否重新变强。
- **弱势反弹**：中期仍弱，只是短期反弹，不要当主线。
- **确认后退潮**：曾经是主线，但已经明显降温。
- **重新升温**：退潮后重新出现修复迹象，但不能一步恢复为强主线。

### 宽度

宽度指行业内有多少股票站上 MA20。

简单理解：

- 宽度高：行业内不是一两只股票在撑，扩散更好。
- 宽度低：可能只是少数个股表现，主线质量不足。

真正的大级别主线，通常不是只有一个龙头涨，而是会出现行业内多层级载体一起修复或走强。

## 5. 主线变化复核怎么读

这是日报里最重要的模块之一。

它不是看今天谁涨得多，而是比较：

- T-1：上一交易日
- T-3：近 3 个交易日前的复核锚点
- T-5：近 5 个交易日前的复核锚点

这个模块回答的是：

**这条主线是在延续、晋级、加速、分歧、降级、退潮，还是只是弱势反弹？**

重点看两列：

- **生命周期判断**
- **处理**

常见判断含义：

| 生命周期判断 | 你应该怎么理解 |
|---|---|
| 延续确认 | 之前强，现在仍然强，继续观察其持续性 |
| 晋级确认 | 从低级别向更高级别发展，值得提高关注 |
| 加速升温 | 变强速度较快，但也要警惕短期过热 |
| 分歧观察 | 中期还没坏，但短期出现不一致 |
| 降级预警 | 强度、宽度或回撤开始变差 |
| 确认后退潮 | 方向已经明显走弱，主要做风险复核 |
| 退潮延续 | 仍未修复，降为低频观察 |
| 企稳重估 | 从退潮里出现修复，但还不能直接恢复为主线 |
| 弱势反弹失败 | 反弹没有升级，继续降低关注 |

这部分能帮助你避免一个大坑：

**不要把每天涨幅靠前的行业都当成新主线。真正的主线需要连续性。**

## 6. 退潮与风险怎么读

退潮模块不是为了吓人，而是为了防止研究框架被“昨日强势记忆”绑架。

重点看三类情况：

1. **曾经强，现在明显回撤**
   这类方向最容易让人误以为“调整后还有机会”，但日报会先标为退潮或风险复核。

2. **短期大跌，宽度快速下降**
   说明行业内部扩散变差，不只是单只股票的问题。

3. **退潮后的单日反弹**
   单日反弹不能直接恢复为候选主线，必须先进入“企稳重估”。

如果一个行业进入退潮，不代表它永远不能研究，而是意味着它必须重新走完修复、升温、候选、确认的过程。

## 7. 主线载体摘要怎么读

载体摘要不是“选股池”，而是回答：

**如果这条主线真的成立，市场通常会通过哪些载体表达？**

常见载体有四类：

- **ETF / 行业指数**
  用来观察行业整体强弱，适合判断主线本身是否扩散。

- **中军龙头**
  通常市值较大、流动性更好，更适合观察机构资金和行业中枢。

- **弹性龙头**
  波动更大，容易表现更强，也更容易过热。

- **风险复核标的**
  不是机会提示，而是提醒这些标的存在估值、过热、回撤、行业退潮等问题。

阅读时要先看“载体类型”和“行动约束”，不要只看分数或强度。

基本面在这里的作用是排雷和解释，不是精细打分。比如：

- 估值极高：说明不适合只因为强势就乐观。
- 盈利承压：说明行业逻辑可能主要来自情绪或政策，而不是业绩。
- 质量较稳：说明它更适合作为中军载体观察。
- 财务数据待补：说明当前不能得出安全边际结论。

## 8. 明日复核清单怎么用

明日复核清单是第二天最应该对照看的内容。

它的作用不是预测明天会怎样，而是提前写下：

- 哪些主线需要验证延续？
- 哪些主线需要确认是否晋级？
- 哪些退潮方向不能因为单日反弹就恢复关注？
- 哪些弱势反弹如果不继续改善，就应该降级？

建议你每天读完日报后，只记 3-5 条复核重点。

第二天再看时，不要问“涨了吗”，而要问：

- 昨天判断的生命周期有没有变化？
- 宽度有没有改善？
- 回撤有没有扩大？
- 龙头状态有没有破坏？
- 这条主线是在延续，还是只是反弹？

这样用，日报才会从“每天一张榜单”变成“连续研究系统”。

## 9. 疑似漏报复核怎么看

疑似漏报模块用于发现：

**某些行业 60 日涨幅较高，但没有进入主线榜。**

它不是机会提示，而是框架审计。

常见原因包括：

- 行业分类太粗，细分主线被稀释。
- 少数个股很强，但行业宽度不足。
- 60 日表现还不错，但近期已经明显回撤。
- 曾经强过，但现在进入退潮或低频监控。

这部分的价值是提醒系统继续迭代，而不是提示你去追漏网之鱼。

## 10. 每日 10 分钟阅读流程

如果时间少，按这个流程：

1. 看今日结论卡片：市场偏强、偏观察还是偏弱？
2. 看市场环境：今日分数和昨日相比是上升还是下降？
3. 看主线变化复核：哪些行业晋级，哪些降级？
4. 看退潮风险：有没有原来强的方向开始坏？
5. 看明日复核清单：记下 3 条明天要验证的事。

这个流程适合日常快速复盘。

## 11. 每周 30 分钟复盘流程

如果做周末复盘，建议这样看：

1. 连续打开最近 5 份日报。
2. 只看主线变化复核，不看个股。
3. 找出连续出现“延续确认”或“晋级确认”的行业。
4. 剔除持续“弱势反弹失败”和“退潮延续”的行业。
5. 对剩下的行业做人工研究：政策、产业、业绩、估值、龙头状态。

周复盘的目标不是找明天最强的行业，而是找：

**哪些方向已经表现出连续性，可能具备中高级别主线特征。**

## 12. 最常见的五个误读

### 误读一：分数高就是值得行动

不是。分数高只代表研究优先级较高，还要看市场环境、生命周期、估值温度、宽度和风险标签。

### 误读二：C 级没有价值

不一定。C 级里可能有早期升温方向，也可能只是弱势反弹。关键看后续能否晋级。

### 误读三：退潮后反弹就是恢复主线

不是。退潮后的单日反弹要先进入企稳重估，不能直接恢复候选或确认。

### 误读四：载体池就是选股池

不是。载体池只是说明主线可能通过哪些标的体现，用于研究分层，不是个股推荐。

### 误读五：日报要每天给出明确机会

不是。很多时候最有价值的结论是：

**市场环境不支持进攻，主线不清晰，继续观察。**

这不是无效结论，反而是保护研究纪律的核心。

## 13. 最佳使用方式

这份日报的最大价值来自连续阅读，而不是单日阅读。

建议你每天只做三件事：

1. 记录今日市场环境状态。
2. 记录主线生命周期变化。
3. 记录明日需要复核的关键行业。

一段时间后，你会得到一条更清晰的路径：

```text
市场环境改善
→ 行业从升温进入候选
→ 候选主线持续扩散
→ 宽度和龙头状态保持
→ 主线获得延续确认
→ 再进入人工行业研究和载体研究
```

这才是本系统真正要捕捉的东西。

## 14. HTML 日报怎么打开

当前 HTML 示例日报位于：

```text
reports/daily_review/a_share_daily_review_2026-06-01.html
```

Markdown 原文位于：

```text
reports/daily_review/a_share_daily_review_2026-06-01.md
```

HTML 更适合日常阅读，Markdown 更适合追溯完整文本和给其他 agent 接管。

## 15. 一句话总结

每天读日报时，不要问：

> 今天哪个板块最强？

而要问：

> 在当前市场环境下，哪些行业正在表现出可持续的主线生命周期，哪些只是短期反弹，哪些已经退潮？

只要坚持这个读法，这份日报的作用就不是追热点，而是帮你建立一套稳定的主线研究纪律。
