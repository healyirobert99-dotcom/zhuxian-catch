# 港股通主线研究系统：项目规则文档

生成日期：2026-06-02

## 1. 项目定位

本项目定位为：

**港股通主线识别与研究闭环系统。**

它不是港股选股器，也不是交易信号系统。核心目标是帮助普通大陆投资者在合规可参与范围内，识别港股市场中具有持续性的行业、主题和龙头载体。

系统只回答四个问题：

1. 当前港股市场环境是否支持主线扩散；
2. 港股通范围内哪些行业或主题正在形成持续性；
3. 这些主线是否得到南向资金、指数环境和龙头权重股确认；
4. 哪些方向值得继续研究，哪些方向已经退潮或只是短期反弹。

系统明确不做：

- 不输出买入、卖出、建仓、加仓、减仓、清仓等交易指令；
- 不做个股短线择时；
- 不猜底；
- 不把股价涨幅榜当作主线；
- 不把高波动小票当成主线核心；
- 不把港股全市场垃圾流动性小票纳入第一版。

## 2. 适用范围

第一版只覆盖：

```text
港股通标的
```

原因：

- 这是大陆普通投资者最主要、最合规、最可执行的港股参与范围；
- 港股全市场中大量小票成交稀薄，容易污染主线判断；
- 港股主线经常由龙头权重股驱动，港股通已经覆盖大部分核心资产；
- 先把可执行范围跑稳，比一开始追求全市场覆盖更重要。

后续可扩展：

```text
恒生综合指数成分股
恒生行业指数
港股全市场高流动性股票
港股 ETF / REITs
```

## 3. 与 A 股项目的区别

港股不能直接照搬 A 股主线日报。

关键差异：

| 维度 | A 股 | 港股 |
|---|---|---|
| 核心资金 | 内资、公募、游资、北向 | 南向资金、外资、机构、回购 |
| 主线表现 | 板块扩散明显，题材轮动快 | 龙头权重股驱动更明显 |
| 流动性 | 中小票也可能活跃 | 大量小票长期低流动性 |
| 指数影响 | 全 A、沪深300、中证800 | 恒生指数、恒生科技、国企指数 |
| 估值特征 | 情绪溢价更强 | 折价、流动性折价、政策折价明显 |
| 主线确认 | 行业宽度 + 题材扩散 | 龙头确认 + 南向资金 + 指数环境 |

因此港股版系统要更重视：

- 龙头权重股；
- 南向资金；
- 成交额集中度；
- 恒生科技 / 恒生指数 / 国企指数；
- 估值修复与政策折价变化；
- 行业 beta，而不是大量个股普涨。

## 4. 核心框架

系统主流程：

```text
港股市场环境
→ 港股通行业 / 主题主线识别
→ 南向资金与指数环境确认
→ 主线载体分层
→ 生命周期复核
→ 每日日报
```

## 5. 市场环境层

市场环境用于判断当前港股是否适合研究进攻型主线。

建议观察指标：

```text
恒生指数 20/60 日趋势
恒生科技指数 20/60 日趋势
恒生国企指数 20/60 日趋势
港股通标的上涨比例
港股通标的站上 MA20 比例
港股通标的站上 MA60 比例
港股通成交额变化
南向资金净流入
南向资金连续净流入天数
创新高 / 创新低数量
大市成交额是否放大
```

环境分建议：

| 环境分 | 状态 | 含义 | 研究约束 |
|---:|---|---|---|
| 0-29 | 防守 | 指数弱、宽度差、资金不配合 | 只做退潮监控 |
| 30-44 | 弱观察 | 局部修复，但市场承接不足 | 只观察，不扩大研究池 |
| 45-54 | 偏观察 | 有结构机会，但持续性待验证 | 早期主线优先复核 |
| 55-69 | 中性偏强 | 指数和资金环境改善 | 可重点研究主线载体 |
| ≥70 | 强环境 | 主线扩散、资金配合、指数共振 | 主线研究价值提高 |

注意：

环境分只影响研究优先级，不生成交易许可。

```text
黄金 / 有色
消费

```

### 6.2 主题主线

例如：


## 7. 主线生命周期

生命周期阶段建议沿用 A 股项目思想，但做港股适配：

```text
低频监控
企稳重估
重新升温
C级结构修复
候选主线
B级主线
A级主线
确认后分歧
确认后退潮
退潮主线
```

关键判断指标：

```text
5日 / 20日 / 60日收益
相对恒生指数超额收益
相对恒生科技指数超额收益
60日峰值回撤
站上 MA20 比例
站上 MA60 比例
成交额放大倍数
龙头股趋势状态
南向资金是否配合
```

## 8. 港股特有确认条件

港股主线不能只看行业平均涨幅。

必须增加三类确认：

### 8.1 龙头确认

每个主线至少维护 3-8 个核心载体。

例如：

```text
互联网：腾讯、阿里、美团、京东、小米
创新药：药明生物、药明康德、信达生物、康方生物、百济神州
电信运营商：中国移动、中国电信、中国联通
能源：中国海洋石油、中国石油、中国神华
半导体：中芯国际、华虹半导体、上海复旦
```

龙头确认看：

```text
是否站上 MA20 / MA60
20日收益是否跑赢指数
成交额是否放大
是否带动同组股票
是否没有明显破位
```

### 8.2 南向资金确认

南向资金是港股通项目的核心变量。

观察：

```text
当日南向净流入
近5日南向净流入
近20日南向净流入
南向成交占比
南向资金集中买入行业
南向资金是否流入主线龙头
```

南向资金不作为单独买点，只作为主线可信度确认。

### 8.3 指数共振确认

至少观察：

```text
恒生指数
恒生科技指数
恒生国企指数
港股通指数
```

如果主线强，但指数环境极弱，应降级为“局部强势 / 仅观察”。

## 9. 主线载体分层

港股项目不追求从同板块内精细选股，而是给出载体分层。

建议分为：

```text
ETF / 指数载体
中军龙头
弹性龙头
高股息载体
风险复核标的
```

展示字段：

```text
主线
代码
名称
载体类型
趋势状态
南向资金状态
估值温度
流动性状态
研究优先级
行动约束
```

注意：

“研究优先级”不等于“可以买”。

## 10. 基本面角色

港股基本面模块只做三件事：

1. 板块支撑判断；
2. 个股排雷；
3. 载体解释。

不做复杂个股加权打分。

建议标签：

```text
高股息稳定
盈利改善
盈利承压
亏损/不可比
估值偏低
估值正常
估值偏高
估值极高
流动性折价
政策折价
财务数据待补
```

港股尤其要注意：

```text
低估值不等于安全；
折价可能长期存在；
高股息要看分红持续性；
创新药和科技股更看产业趋势与业绩兑现路径；
地产、物业、金融等低估值行业需要额外风险标签。
```

## 15. 初始回测方向

第一阶段不做个股收益回测。

只验证：

```text
早期主线信号出现后，
对应行业 / 主题 / 龙头篮子在 20/40/60 日内，
是否跑赢恒生指数或恒生科技指数。
```

观察指标：

```text
20日超额收益
40日超额收益
60日超额收益
20/40/60日最大峰值空间
到达峰值所需交易日
信号后最大回撤
南向资金是否持续配合
```

样本建议：

```text
2021-2026 年港股通历史
按年度分片验证
单独观察 2024-2026 的结构性修复行情
```

## 17. 最终原则

港股通主线系统的核心不是“找小票”，而是：

```text
识别港股通范围内，指数、南向资金、龙头权重股和行业主题共同指向的中级别主线。
```

如果一句话概括：

> A 股系统偏“主线扩散识别”，港股系统应偏“主线龙头与南向资金共振识别”。
