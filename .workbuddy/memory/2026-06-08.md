# 2026-06-08 工作日志

## 催化剂层代理回测验证

### 执行内容
- 用概念 pct_change 绝对值作为市场关注度代理变量
- 对 early_core_env45 2653 个信号分组对比：有催化代理 vs 无催化代理
- 脚本: `scripts/backtest_catalyst_proxy.py`

### 核心发现（反直觉）

|pct|阈值越高，胜率越低：
- |pct|>=2%: 有催化胜率61.4% vs 无催化67.7%（-6.4%）
- |pct|>=3%: 有催化胜率56.9% vs 无催化67.6%（-10.8%）
- |pct|>=4%: 有催化胜率55.3% vs 无催化67.3%（-12.1%, p=0.006 显著）
- |pct|>=5%: 有催化胜率43.6% vs 无催化67.3%（-23.7%, p=0.030 显著）

但峰值更高（5.65% vs 3.70%），说明收益双峰化——少数大涨，多数失败。

### 控制 ret5 后
同一 ret5 水平下，有催化仍全部跑输（-3.8% ~ -18.6%），说明概念波动有独立负效应。

### 环境翻转
偏观察(45-54)环境下有催化反而正效应（所有ret5区间全胜 +12%~+33%），说明弱市中概念波动是正面信号。

### 结论
- 不是未来函数问题（概念指数 pct_change 是 THS 历史实时计算的）
- 概念大幅波动 = 价格已透支
- 环境 >=55 且概念波动 >=3% 不宜追入
- 偏观察环境下此规则不适用

## 企稳重估信号升级速度回测

### 回测结论（1300条 early_core_env45）
- 97.8% 的企稳信号最终升至 C 级以上，但中位需 **37 天**
